# スタイルファイルの構成

`public/styles.css` から読み込む分割済みのスタイル定義です。役割ごとに編集先を確認してください。

- `base.css`: ルート変数、リセット、背景
- `header.css`: トップバーやチップ類
- `workspace.css`: キャンバス、デスクトップ、アプリカード、コンテキストメニュー
- `panels.css`: サイドパネル、設定フォーム、ステータス表示
- `quick-actions.css`: クイックアクションやそのレスポンシブ調整
- `windowing.css`: ドック、ウィンドウ、関連アニメーション
- `components.css`: 汎用コンポーネント（リスト、ボタンなど）
- `responsive.css`: キャンバスのブレークポイント調整
