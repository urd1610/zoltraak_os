# Claude Code と GLM Coding Plan - スライドアプリ ユーザーガイド

## アプリケーション概要

**Claude Code** と **GLM Coding Plan** の活用方法を5つのスライドで説明する、Kamui OS ベースのスライド表示アプリケーションです。

### 主なコンテンツ
1. **スライド1**: Claude Code と GLM Coding Plan（タイトル）
2. **スライド2**: Claude Code の機能紹介
3. **スライド3**: GLM Coding Plan のメリット（3倍パフォーマンス）
4. **スライド4**: セットアップ手順（3ステップ）
5. **スライド5**: 実装活用例とベストプラクティス

## 操作ガイド

### キーボード操作
| キー | 機能 |
|------|------|
| **→ 右矢印** | 次のスライドへ |
| **← 左矢印** | 前のスライドへ |
| **Esc** | サイドバー折りたたみ/展開 |

### マウス操作
| 操作 | 機能 |
|------|------|
| **左矢印ボタン** | 前のスライドへ |
| **右矢印ボタン** | 次のスライドへ |
| **スライド一覧項目クリック** | 指定スライドへ移動 |
| **◀ ボタン（サイドバー）** | サイドバー折りたたみ/展開 |
| **ダウンロードボタン** | 全スライドを JPEG として PDF に変換 |

### サイドバー機能
- **スライド一覧**: 全5スライドの概要を表示
- **クリック移動**: 任意のスライドをクリックして移動
- **折りたたみ**: 「◀」ボタンで サイドバーを最小化（矢印キーで操作可能）
- **ダウンロード**: 「⬇」ボタンで全スライドを JPEG 画像として PDF に保存

## スライドコンテンツ詳細

### スライド 1: タイトル
- **内容**: Claude Code と GLM Coding Plan の紹介
- **ビジュアル**: ターミナルウィンドウ、紫→青グラデーション、稲妻アイコン
- **メッセージ**: 3倍のパフォーマンス実現

### スライド 2: Claude Code の機能
- **内容**: 自然言語コマンド、コード理解、ルーチンタスク自動化、説明機能、Git ワークフロー
- **ビジュアル**: ターミナルプロンプト + 5つの機能アイコン
- **主要機能**: コード理解、自動化、説明、Git、スピード

### スライド 3: GLM Coding Plan のメリット
- **内容**: より多くのトークン、安定した信頼性、より速いコーディング、スマートなデバッグ
- **ビジュアル**: 中央に「3倍」の数字、4つのメリットアイコン
- **キーメッセージ**: パフォーマンス × 信頼性 × スマート

### スライド 4: セットアップ手順
- **ステップ 1**: `npm install -g @anthropic-ai/claude-code`
- **ステップ 2**: Z.AI でAPIキー取得 → 環境変数設定
- **ステップ 3**: `claude` コマンドで実行開始
- **ビジュアル**: 左から右へのステップフロー表示

### スライド 5: 実装活用例
- **用途 1**: ルーチンタスク自動化（チェックボックス）
- **用途 2**: 複雑なバグ調査（虫眼鏡）
- **用途 3**: 大規模リファクタリング（ハンマーレンチ）
- **用途 4**: マルチファイル変更管理（フォルダ）

## トラブルシューティング

### 問題: ダウンロードボタンが機能しない
**原因**: jsPDF ライブラリ CDN エラー  
**解決**: ネットワーク接続確認、キャッシュクリア

### 問題: スライドが表示されない
**原因**: 画像 URL が無効  
**解決**: URL を確認、メタデータと照合

### 問題: キーボード操作が効かない
**原因**: ブラウザフォーカス外  
**解決**: スライドエリアをクリック後に矢印キーを使用

### 問題: サイドバー項目が更新されない
**原因**: alt 属性が欠落  
**解決**: 全 img タグに alt 属性を設定

## ライセンスと利用規約

このアプリケーションは **Kamui OS テンプレート** をベースにしており、スライドコンテンツは z-ai.md ドキュメントに基づいています。

**禁止事項**:
- ベース HTML/CSS/JS の改変
- ダウンロードボタンの削除・改造
- DOM 構造・クラス名の変更

**推奨される拡張**:
- メタデータ JSON の画像 URL 更新
- alt 属性の拡張説明
- アイコン SVG のカラーバリエーション
